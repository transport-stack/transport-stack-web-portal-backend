# Transport Stack Web Portal Backend

Backend services for the Transport Stack web portal. This project provides the core API and data management layer for the Transport Stack ecosystem.

## Tech Stack

- Java / Spring Boot
- RESTful APIs

## Getting Started

### Prerequisites

- Java 17+
- Maven or Gradle

### Setup

```bash
git clone https://github.com/transport-stack/transport-stack-web-portal-backend.git
cd transport-stack-web-portal-backend
```

Follow the project's build instructions to compile and run the application.
